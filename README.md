# LetsTry – Exercices Pratiques de Développement Web

Un ensemble d'exercices pratiques pour apprendre le développement web avec HTML, CSS et JavaScript.

---

## 📂 Structure des exercices

```
LetsTry/
├── exercice1/   Page de profil personnel (HTML + CSS)
├── exercice2/   Barre de navigation responsive (HTML + CSS + JS)
├── exercice3/   Calculatrice (HTML + CSS + JS)
├── exercice4/   Liste de tâches (HTML + CSS + JS)
└── exercice5/   Mise en page Flexbox & Grid (HTML + CSS)
```

---

## Exercice 1 – Page de profil personnel

**Objectif :** Créer une page de profil personnel structurée en HTML et stylisée en CSS.

**Concepts abordés :**
- Sémantique HTML5 (`header`, `main`, `section`, `footer`)
- Mise en forme de base avec CSS (couleurs, typographie, espacement)
- Listes et liens

**Fichiers :** `exercice1/index.html`, `exercice1/style.css`

---

## Exercice 2 – Barre de navigation responsive

**Objectif :** Construire une barre de navigation avec un menu qui s'adapte aux écrans mobiles (menu hamburger).

**Concepts abordés :**
- Flexbox pour l'alignement horizontal
- Media queries pour le responsive design
- Manipulation du DOM en JavaScript (ouverture/fermeture du menu)

**Fichiers :** `exercice2/index.html`, `exercice2/style.css`, `exercice2/script.js`

---

## Exercice 3 – Calculatrice

**Objectif :** Créer une calculatrice fonctionnelle avec les quatre opérations de base.

**Concepts abordés :**
- Grille CSS (`grid`) pour la disposition des boutons
- Gestion des événements JavaScript (clic et clavier)
- Manipulation du DOM (affichage du résultat)
- Validation des entrées

**Fichiers :** `exercice3/index.html`, `exercice3/style.css`, `exercice3/script.js`

---

## Exercice 4 – Liste de tâches (Todo List)

**Objectif :** Développer une application de liste de tâches avec ajout, suppression, filtrage et persistance des données.

**Concepts abordés :**
- Manipulation du DOM (création et suppression d'éléments)
- Gestion des événements JavaScript
- `localStorage` pour la persistance des données
- Filtrage dynamique de contenu

**Fichiers :** `exercice4/index.html`, `exercice4/style.css`, `exercice4/script.js`

---

## Exercice 5 – Mise en page Flexbox & Grid

**Objectif :** Construire une page complète utilisant Flexbox et CSS Grid pour la mise en page.

**Concepts abordés :**
- CSS Flexbox (header, hero, footer)
- CSS Grid (grille d'articles avec carte mise en avant, galerie d'images)
- Variables CSS (`custom properties`)
- Design responsive avec media queries

**Fichiers :** `exercice5/index.html`, `exercice5/style.css`

---

## Comment utiliser ces exercices

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/gitamadoudiene/LetsTry.git
   cd LetsTry
   ```

2. Ouvrir le fichier `index.html` de l'exercice choisi dans votre navigateur.

3. Modifier le code, expérimenter et apprendre !

> **Conseil :** Essayez d'abord de réaliser chaque exercice vous-même avant de consulter le code fourni.
